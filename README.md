# Test Management System
## Overview
This project is designed to manage and execute tests. It provides a scalable and efficient backend framework for handling test execution and results analytics.
## Features
* Test Creation
* Test Execution
* Results Analytics
## Requirements
* Node.js
* Express.js
* MySQL
## Installation
1. Clone the repository
2. Install the dependencies using `npm install`
3. Create a database and update the environment variables in `.env`
4. Start the server using `npm start`
